* 프로젝트 회고

>
교육 수료 중, 가장 큰 프로젝트가 아니었나 싶다. 생각보다 오류를 많이 잡아야만 했고, 
각 디바이스에 맞춰 출력하기 위해서 고민을 많이 했던 것으로 기억한다. 
‘예약하기’ 기능을 구현할 때도 빈방 찾기를 구현하는 쿼리를 찾느라 애를 썼었다. 
가장 어려웠던 것은 무엇보다 웹 디자인이었다. 웹 디자인을 좀 더 꼼꼼히 공부해서 기본적인 레이아웃을 갖추어야 겠다.
